I made this extension to help control my procrastination problem. Although there are other extensions similar to it, I decided to make my own to improve my coding skills. I also added features that some other website blocking extensions do not have, such as: the ability to add reasons for blocking the site, the ability to permablock sites and the ability to manually unblock a site that you have blocked by entering your password.

If you like this extension, and found it useful, please consider donating below. This will allow me to push updates regularly, and help me make more things like it.
